Не працює Face ID на iPhone: чому і що можна зробити самому

Майстерня SPARK·23 липня 2026·8 хв читання

Телефон перестав упізнавати вас в обличчя, а замість розблокування просить код? Або спливає напис «Face ID недоступний»? Причин може бути з десяток — від банального чохла, що перекрив датчик, до поломки блока TrueDepth після падіння. Гарна новина: приблизно половина випадків вирішується за п'ять хвилин удома. Розберемо все по порядку — від простого до складного, щоб ви не понесли в сервіс те, що лагодиться власними руками.

Коротко
  • Face ID працює через блок TrueDepth — крихітні датчики в «чубчику»; вони крихкі й бояться ударів, вологи та бруду.
  • Половина випадків лікується вдома: зняти чохол-плівку з датчиків, перезавантажити телефон, наново налаштувати Face ID, оновити iOS.
  • «Face ID недоступний» часто вилазить після заміни екрана дешевою копією — пошкоджується або відключається шлейф датчика.
  • Якщо після падіння чи води виходить «Не вдалося активувати Face ID» — це вже апаратна поломка датчика, удома не лагодиться.
  • Точну причину покаже діагностика: софт це, шлейф після заміни екрана чи сам блок TrueDepth.

Як узагалі працює Face ID

Face ID — це не звичайна фронтальна камера. За розблокування по обличчю відповідає блок TrueDepth: набір мініатюрних датчиків у вирізі екрана, які будують об'ємну карту обличчя з тисяч невидимих точок.

  • Проектор точок (dot projector) малює на обличчі сітку з ~30 000 інфрачервоних точок.
  • Інфрачервона камера зчитує цю сітку й будує 3D-модель обличчя.
  • Датчик освітленості (flood illuminator) підсвічує обличчя в темряві, тому Face ID працює вночі.

Усі ці датчики крихітні й з'єднані з платою тонкими шлейфами. Звідси й головна вразливість: сильний удар, потрапляння вологи або необережне втручання у верхню частину екрана легко виводять блок з ладу. Причому Apple жорстко прив'язує TrueDepth до конкретної плати — просто «переставити датчик з іншого телефона» не вийде.

Програмні причини — лагодяться за п'ять хвилин удома

Перш ніж думати про ремонт, варто виключити найчастіше й найбезпечніше. Дуже багато «поломок» Face ID — це перекритий датчик або тимчасовий збій системи.

  • Чохол або захисне скло перекрили «чубчик». Товстий чохол, криво наклеєне скло чи плівка з вирізом не по розміру закривають проектор точок — Face ID просто не бачить обличчя.
  • Бруд і запотівання. Пил, відбитки чи конденсат на датчиках заважають скануванню — достатньо протерти верх екрана.
  • Тримаєте телефон надто близько або під гострим кутом. Face ID хоче бачити обличчя цілком з відстані 25–50 см.
  • Тимчасовий збій iOS. Після оновлення або зависання функція іноді «відвалюється» до перезавантаження.
  • Не налаштований другий зовнішній вигляд. Якщо ви носите окуляри, відростили бороду чи часто в масці — додайте «Альтернативний зовнішній вигляд» у налаштуваннях.
Прогоніть короткий чек-лист із блока «Що зробити» нижче: зняти чохол → протерти датчик → перезавантажити → налаштувати Face ID наново → оновити iOS. Якщо після цього обличчя розпізнається — проблема була софтовою, і в сервіс іти не потрібно.

«Face ID недоступний» після заміни екрана

Окрема й дуже часта історія: Face ID працював, ви поміняли розбитий екран у стороннього майстра — і він зник. Річ у тім, що при заміні дисплея шлейфи Face ID проходять поряд із модулем, і їх легко пошкодити або необережно відключити.

  • Пошкоджений шлейф проектора точок при знятті старого екрана — найчастіша причина «зниклого» Face ID після ремонту.
  • Не перенесли або пошкодили рідний датчик при встановленні нового модуля.
  • Дешева копія дисплея з неякісним шлейфом або екранування заважає роботі блока.

Висновок простий: заміна екрана сама по собі не повинна ламати Face ID, якщо працювати акуратно й не пошкоджувати шлейфи. Саме тому важливо, кому ви віддаєте телефон — відновити Face ID після кривого ремонту на порядок складніше й дорожче, ніж зробити заміну правильно з першого разу. Якщо Face ID зник одразу після стороннього ремонту, несіть на діагностику — часто це вирішувано.

Апаратні причини — лише сервіс

Якщо програмні кроки не допомогли й телефон не був у чужому ремонті, найімовірніше постраждало «залізо». Зазвичай цьому передує конкретна подія.

  • Падіння або удар. Від струсу відходять чи рвуться шлейфи, тріскається сам проектор точок. Часто Face ID відмовляє не одразу, а через кілька днів після падіння.
  • Волога. Навіть без прямого потрапляння у воду конденсат усередині блока TrueDepth окислює контакти. Якщо телефон побував у воді, Face ID — один з перших, хто відмовляє.
  • Повідомлення «Не вдалося активувати Face ID» або «Потрібне обслуговування Face ID». Це прямий знак апаратної несправності датчика або його шлейфа.
Ремонт блока TrueDepth — це мікропайка й робота з датчиком, прив'язаним до плати. Таке завдання бере не кожен сервіс, і до початку робіт чесний майстер завжди каже, чи реально відновити Face ID саме у вашому випадку, чи дешевше залишити розблокування за кодом.

Що робити покроково

Зберемо все разом. Порядок дій — від безкоштовного й швидкого до того, що потребує майстра:

  1. Зніміть чохол і захисне скло, протріть верх екрана, перевірте налаштування Face ID.
  2. Перезавантажте телефон і спробуйте знову.
  3. Скиньте й налаштуйте Face ID наново за хорошого освітлення, додайте альтернативний зовнішній вигляд.
  4. Оновіть iOS до останньої версії.
  5. Не допомогло або є напис «Face ID недоступний / не активується» — на діагностику. Там визначать, що саме: шлейф після заміни екрана чи сам блок TrueDepth.

Діагностика в SPARK безкоштовна: ми під збільшенням перевіримо датчики й шлейфи та скажемо, чи лагодиться Face ID у вашому випадку, скільки це коштує і чи є сенс — без нав'язування зайвого.

Коротко про головне

Підсумок: не поспішайте нести iPhone у сервіс, поки не пройшли простий чек-лист — зняти чохол, протерти датчик, перезавантажити, налаштувати Face ID наново й оновити iOS. Це закриває більшу частину випадків безкоштовно. Якщо Face ID зник після заміни екрана — найчастіше річ у пошкодженому шлейфі, і це вирішувано. А от напис «Не вдалося активувати Face ID» після падіння чи води означає поломку самого блока TrueDepth — тут потрібна діагностика й мікропайка, і не завжди ремонт виправданий. Принесіть телефон на безкоштовну діагностику — чесно скажемо, що з ним і чи є сенс лагодити.

Face ID недоступний і не лікується сам?

Безкоштовно перевіримо блок TrueDepth і шлейфи під збільшенням, визначимо причину й скажемо, чи реально відновити Face ID саме у вашому випадку.

Часті запитання

Чому Face ID перестав працювати, хоча телефон не падав?
Найчастіше це програмна причина: чохол або скло перекрили датчик, забруднився «чубчик», стався збій iOS або збилися налаштування. Зніміть чохол, протріть верх екрана, перезавантажте телефон і налаштуйте Face ID наново. Якщо не допомогло — потрібна діагностика.
Face ID зник після заміни екрана — це можна полагодити?
Часто так. При заміні дисплея легко пошкодити або необережно відключити шлейф Face ID. У частині випадків функцію відновлюють, переустановивши або замінивши пошкоджений шлейф. Точну відповідь дає діагностика — відновити після кривого ремонту складніше, ніж зробити заміну акуратно одразу.
Що означає «Не вдалося активувати Face ID»?
Це знак апаратної несправності блока TrueDepth або його шлейфа — зазвичай після падіння чи вологи. Програмні кроки тут не допоможуть, потрібна діагностика і, як правило, мікропайка. Іноді відновлення недоцільне, і залишається розблокування за кодом.
Чи можна переставити датчик Face ID з іншого iPhone?
Ні. Apple прив'язує блок TrueDepth до конкретної плати телефона захищеним каналом. Датчик з іншого апарата працювати не буде. Відновлення можливе лише з рідним датчиком через мікропайку, якщо він фізично цілий.
Чи можна користуватися телефоном, якщо Face ID не лагодиться?
Так. Розблокування за код-паролем, оплата підтвердженням коду й усі інші функції працюють у штатному режимі. Face ID — зручність, а не критична для роботи телефона система, тому багато хто залишає код-пароль, якщо ремонт датчика недоцільний.
